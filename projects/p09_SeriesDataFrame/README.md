# Project Overview

This project introduces the fundamentals of working with Pandas DataFrames and Series using the Titanic dataset from Seaborn. The analysis covers dataset inspection, shape checking, data types, descriptive statistics, and the difference between selecting single columns as a Series versus multiple columns as a DataFrame.

The project also explores basic dataset structure analysis, helping build familiarity with core Pandas operations commonly used in data cleaning and exploratory data analysis (EDA).
